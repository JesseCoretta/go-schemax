# Security Policy

It is unlikely this package would be a factor in any security-related scenario.  This package is an abstract builder of RFC 4512 constructs.  This package is a glorified sandbox in that context.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| >=2.0.0 | ✅                 |
| < 2.0.0 | ❌                 |
